This is the data directory. It has the following directory structure:
data/
- original
- derived

The data in the data/original directory is:
 - BMX_D.csv, a csv file containing body measurement data for a list of patient ids
 - DEMO_D.XPT, an xpt file containing demographics data for the same patient ids
 - accel, a directory of accelometer readings with one file per patient id
